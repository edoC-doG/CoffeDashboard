# **App Name**: CafeFlow

## Core Features:

- Inventory Tracking: Track stock levels of coffee beans, milk, sugar, cups, and lids.
- Sales Recording: Record each sale, including items sold, price, and payment method.
- Reporting: Display charts for inventory, best sellers, total sales
- Order Management: Handle processing orders for beverages

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke a sense of freshness and calm.
- Background color: Light cyan (#E0FFFF), a desaturated version of sky blue for a clean and airy feel.
- Accent color: Steel blue (#4682B4), a darker shade of blue to draw attention to key elements.
- Headline font: 'Open Sans', a clean and modern sans-serif font for headers.
- Body font: 'Roboto', a widely used sans-serif font offering excellent readability for descriptions and data displays.
- Use line icons from a set like Material Design Icons, providing clarity and a modern aesthetic.
- Responsive grid layout optimized for desktop and mobile views, utilizing Tailwind CSS for flexible and consistent spacing.
